6.2.1. Оценка биологического значения результатов, получаемых с
помощью программной компоненты compare_aln_3.
Биологический смысл программы compare_aln_3 — в сравнении двух
выравниваний, построенных разными программами. Для оценки
получаемых результатов:
6.2.1.1. Подготовить три множества родственных
последовательностей белков set1, set2, set3 для которых известна
пространственная структура. Источником служат
последовательности из БД MDSA.
6.2.1.2. Для каждого из множеств построить выранивание
последовательностей с помощью двух различных программ из
числа свободно распространяемых (например, muscle, mafft, probcons, dialign
или clustalw). Результаты положить в файлы, чьи названия отражали
бы как множества, так и программы, например: set1_muscle.fasta и
set1_clustal.fasta.
6.2.1.3. Для каждого множества запустить программу compare_aln_3d на
соответсвующей паре выравнивания, для чего выполнить команду
вида
compare_aln_3d compare_aln_3d -a et1_muscle.fasta -b set1_clustal.fasta
и сохранить результат из стандартного потока вывода (stdout)
6.2.1.4. Сравнить определение лучшей программы по результатам
работы compare_aln_3d с экспертным заключением о выравниваниях,
построенных двумя программами. Программная компонента
compare_aln_3d считается выдержавшей проверку, если её результаты во
всех трёх случаях не противоречат экспертным заключениям.
